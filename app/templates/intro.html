Get started with PGP

This website helps you send encrypted messages using PGP.

What is PGP? Is pgp.help secure?

What is PGP?

  • PGP allows you to send encrypted messages. Encrypted messages can only be read by the recipient and no-one else.
  • To send a message you need the recipient’s public key. You will have to ask them to give you this.
  • To receive messages you need a private key. We can generate one for you.

Is pgp.help secure?

This website is not 100% secure! If privacy is important to you then make sure you understand the risks!

  • All the code in this website runs locally inside your browser. No data is ever sent over the internet.
  • However web browsers occasionally have security vulnerabilities, and are often targeted by hackers
  • Additionally since the website itself is loaded over the internet which presents an inherent risk.
  • This website is only intended for casual PGP usage. Do not use this website if privacy is critical for you.

Send a PGP Message

If you want to send an encrypted message and already have their PGP key it’s really simple:

  1. Paste their PGP key into the form.
  2. Type your message.
  3. Copy the encrypted result and send it to your recipient
